# 🧠 TMUX SHORTCUTS — Custom Setup

> **Prefix:** `Ctrl + a`

---

## 🔁 General
| Action | Keys |
|--------|------|
| Prefix | **Ctrl + a** |
| Reload config | `<prefix> + r` |
| Detach session | `<prefix> + d` |
| Command prompt | `<prefix> + :` |
| Toggle status bar | `<prefix> + s` |
| Show help popup | **Ctrl + h** |
| Open Sesh launcher | **Ctrl + Space** |

---

## 🪟 Window Management
| Action | Keys |
|--------|------|
| New window (same CWD) | `<prefix> + c` |
| Rename window | `<prefix> + r` |
| Rename session | `<prefix> + R` |
| Previous window | `<prefix> + C-k` |
| Next window | `<prefix> + C-j` |
| Next window | `<prefix> + z` |
| Last window (MRU) | `<prefix> + a` |

---

## 🧩 Pane Management
| Action | Keys |
|--------|------|
| Split vertical | `<prefix> + v` |
| Split horizontal | `<prefix> + h` |
| Focus next pane | `j` |
| Focus prev pane | `k` |
| Swap pane down | `<prefix> + C-o` |
| Select next pane (cycle) | `<prefix> + Tab` |
| Toggle zoom | `<prefix> + g` |

---

## 🔪 Killing & Closing
| Action | Keys |
|--------|------|
| Kill pane | `<prefix> + x` |
| Kill window | `<prefix> + X` |
| Kill other windows | `<prefix> + C-x` |
| Kill full session | `<prefix> + Q` |

---

## 🔁 Session / Window Operations
| Action | Keys |
|--------|------|
| Link window from another session | `<prefix> + L` |
| Merge all windows into another session | `<prefix> + C-u` |
| Show active sessions | `<prefix> + s` |
| Next session | `<prefix> + )` |
| Previous session | `<prefix> + (` |

---

## 👁️ Monitoring / Notifications
| Action | Keys |
|--------|------|
| Toggle activity monitor | `<prefix> + m` |
| Toggle silence monitor | `<prefix> + M` |

---

## 📋 Copy Mode (vi-style) & Scroll
| Action | Keys |
|--------|------|
| Enter copy mode | `Alt + ↑` |
| Scroll up/down | `Alt + ↑ / Alt + ↓` |
| Page up/down | `PgUp / PgDown` |
| Half page up/down | `Alt + PgUp / Alt + PgDown` |
| Yank selection | `y`, `Enter`, `Ctrl + j` |
| Copy on mouse drag end | mouse drag & release |

---

## 🧭 Navigation Summary
| Category | Keys |
|----------|------|
| Windows | `<prefix> + C-j / C-k / a` |
| Panes | `j / k` |

---

## 🧰 Tools
| Tool | Keys |
|------|------|
| Sesh popup | **Ctrl + Space** |
| Keymap help (this file) | **Ctrl + h** |

---

### Tips
- Window and pane numbering starts at **1** (human-friendly)
- Automatic renaming is **disabled** — meaningful names stay
- Mouse support enabled
- Catppuccin theme active
